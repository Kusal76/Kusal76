## 💻 Featured Projects

### [🧠 VectorDB Engine — Custom AI Search & RAG Pipeline](https://github.com/Kusal76/custom-vectordb-engine)
*Python, FastAPI, NumPy, Ollama, Docker*
* Engineered a production-grade **Vector Database and RAG engine entirely from scratch** — zero reliance on managed services like Pinecone or ChromaDB — to demonstrate deep understanding of AI system internals.
* Implemented a custom **HNSW (Hierarchical Navigable Small World)** approximate nearest-neighbor indexing algorithm from the ground up, achieving **sub-10ms query latency** at scale.
* Built a **Hybrid Search** algorithm combining cosine similarity vector retrieval with exact-match keyword scoring to dramatically improve result accuracy for named entities.
* Containerized the full application using Docker with a `host.docker.internal` network bridge to a locally running Ollama instance, enabling fully offline, hardware-accelerated inference with zero data egress.

💻 **[Source Code](https://github.com/Kusal76/custom-vectordb-engine)**
